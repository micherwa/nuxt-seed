# Nuxt3 Seed

## 技术栈

Nuxt3.12 + Vue3.4 + tailwindcss3.4 + pnpm7.13 + node18

## 安装

```bash
# pnpm
pnpm install
```

## 本地启动

本地启动地址： `http://localhost:3000`

```bash
# npm
npm run dev
```

## 打包发布

```bash
# npm
npm run build
```

打包后启动服务

```bash
# npm
npm run preview
```
